# **Portfolio - Gabriel Alves**
Um site de portfólio moderno e ultra-performático, desenvolvido com **SvelteKit** e **Typescript**, combinando **Shadcn/UI** e **TailwindCSS** para uma interface limpa e responsiva. Animações fluidas e interativas foram criadas com **Svelte Motion** e **Svelte Animations**, entregando uma experiência leve, dinâmica e profissional.
